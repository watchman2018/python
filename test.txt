Git基本操作
有新学习的操作就加进去，错误的可以修改

连接远程分支
git remote add origin git@github.test/hellotest.git 

克隆远程分支
git clone 

添加文件
git add
git add .
git add *

提交文件，添加注释
git commit -m ""

推送到远程分支
git push 

如果远程的修改了，需要先fetch或者pull,然后push
git fetch origin master
git pull origin master

删除文件或者文件夹
  1.删除服务器文件,本地保留
git rm --cached useless.log
git commit -m "remove file from remote repository"
git push
此时github上已经不存在了

   2.删除远程useless 文件夹,本地保留
一定要注意，删除文件夹要使用-r 参数
git rm --cached -r useless
git commit -m "remove directory from remote repository"
git push

寻求帮助
git help <verb>
或者
git <verb> --help

回退到指定版本

git reset --hard HEAD^ 回退到上一个版本
git reset --hard 3628164 指定回到某个版本，3628164为commit_id前几位
$ git push origin HEAD --force

查看操作记录
git reflog 记录每一次操作

初始化某个文件夹
git init
